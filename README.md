# Leap Code

A layout for Web Developers

- Windows, see [AutoHotKey](./ahk/)
- Linux, see [xmodmap](./xmodmap/) (experimental)
- Dygma Raise and Dygma Defy, see [dygma](./dygma/)
- Ultimate Hacking Keyboard (UHK), see [uhk](./uhk/)
